# 프로젝트 정리 요약

## 목적

한이음 ICT 멘토링에서 수행한 금융 상품 가격 예측 시스템 프로젝트를 GitHub 포트폴리오와 기록용으로 정리했다. 원본 실험 흐름은 보존하되, 공개 저장소에서 바로 읽을 수 있는 대표 코드와 문서를 추가했다.

## 주요 작업 흐름

1. KRX, 네이버 금융, Kiwoom OpenAPI를 활용해 종목 코드와 일봉 데이터를 수집했다.
2. MySQL/SQLite에 주가 데이터를 저장하고 종목별 테이블을 구성했다.
3. 이동평균, 거래량, 변동성, 추세, 모멘텀 지표를 생성했다.
4. ARIMA, RNN, LSTM, GRU 모델을 실험하고 예측 성능을 비교했다.
5. 네이버 금융 뉴스 제목을 수집해 워드클라우드를 생성했다.
6. PyQt 기반 GUI로 종목 검색, 차트, 예측 결과 표시 화면을 구현했다.

## 공개용 코드 정리 원칙

- 원본 실험 파일은 로컬에 보존했다.
- 공개용 대표 구현은 `src/` 아래에 새로 정리했다.
- 하드코딩된 DB 접속정보는 환경변수 기반 설정으로 대체했다.
- 깨진 주석과 중복 import는 대표 코드에서 제거했다.
- 실행 재현보다 포트폴리오 가독성과 프로젝트 흐름 전달을 우선했다.
