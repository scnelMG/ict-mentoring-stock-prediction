# 공개 안전성 검토

## 공개 기준

이 저장소는 주가 예측 시스템의 구조와 실험 흐름을 보여주는 포트폴리오입니다. 실제 투자 조언, 실거래 시스템, 수익 보장 목적이 아니며, credential, API key, 개인 DB 접속 정보는 공개 대상에서 제외합니다.

## 검토 포인트

- 공개 자료: 샘플 주가 데이터, 수집/모델링/GUI 구조 문서
- 제외 자료: 실제 DB host, password, token, API key, credential 파일

## 리뷰어 안내

평가 대상은 데이터 수집, 기술 지표 생성, RNN 실험, GUI 연결 구조입니다.
